NorskTale Extension Setup Browser bridge for textarea cursor tracking

1
Load Extension in Edge
Open edge://extensions/ in Edge

Enable Developer mode (toggle in sidebar)

Click Load unpacked and select: C:\Users\pette\dev\contexter\NorskTale\acatts-rust\extension
2
Enter Extension ID
Copy the ID shown under the extension in Edge:
3
Save Native Manifest
Click Copy Manifest on the right, then save to: C:\Users\pette\dev\contexter\NorskTale\acatts-rust\extension\com.norsktale.bridge.json Or click Auto-Save to write it directly.
4
Register Native Host
Copy and run the registry commands from the right panel in a terminal (no admin needed — uses HKCU).
5
Restart Edge & NorskTale
Close and reopen Edge, then start NorskTale. Click in a textarea — NorskTale should now track your cursor position.
